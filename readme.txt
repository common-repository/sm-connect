== Description ==

= Connect your blog with Smart Member =

SmartMember is an innovative product created by Chris Record that enables you to rapidly create a membership site and sell access.

This plugin will enable your blog to connect with your account at SmartMember to enable you to import all your WordPress content into Smart Member.

== Installation ==

1. Backup your Wordpress site and database
2. Upload the content inside the zip file to '/wp-content/plugins/' directory OR go to add a new plugin and search for SmartmMmber Connector and install it through Wordpress.
3. Activate the plugin through the 'Plugins' menu in Wordpress
4. Visit SM Connector for scanning and options

== Frequently Asked Questions ==

= How can I use this plugin? =
* Simply generate the connector key in SM Connector. Then, in your SmartMember site got to Integrations and add a Wordpress Integration.
